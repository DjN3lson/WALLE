Proyecto: Mapeo y Movimiento de un Robot en 2D usando Álgebra Lineal
Este proyecto es una demostración sencilla de cómo los vectores y matrices se aplican en la ingeniería de sistemas y mecatrónica, específicamente en el modelado y control del movimiento de un robot móvil.

Descripción
El objetivo es mostrar, en un entorno 2D, cómo se puede simular el movimiento de un robot utilizando conceptos básicos de álgebra lineal. El usuario puede controlar el robot con las flechas del teclado, permitiendo moverlo hacia adelante, hacia atrás y girar a los lados. El proyecto ilustra cómo las operaciones con vectores y matrices permiten calcular la posición y orientación del robot en cada instante.

Características
Simulación del movimiento de un robot en un plano 2D.
Control del robot en tiempo real usando las teclas de flecha.
Aplicación de transformaciones lineales (rotaciones y traslaciones) para actualizar la posición y orientación del robot.
Ejemplo educativo para comprender el uso de álgebra lineal en robótica y automatización.

Requisitos
Lenguaje: Python usando la libreria numpy

Uso
Clona este repositorio.
Ejecuta el archivo principal del proyecto. (visual)
Usa las flechas del teclado para mover el robot en el entorno 2D.
